# owlang.github.io
Personal Github Pages website

See website rendered at: https://owlang.github.io/

## Jekyll development

Install Ruby 3.1.2 and dependencies...
```
gem install jekyll bundler
```

...then test the website by running:
```
bundle install
bundle exec jekyll serve
```
